\subsection*{Introduction}

This super-block compresses the function of the following blocks:
\begin{itemize}
\item Photodiode;
\item Trans-Impedance Amplifier;
\end{itemize}
\noindent
This compression allows for a cleaner code. 

\subsection*{Input Parameters}

\begin{itemize}
	\item Responsivity
	\item Gain
\end{itemize}
